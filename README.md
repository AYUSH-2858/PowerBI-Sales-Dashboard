# Super Store Sales Dashboard

## Overview
Interactive Power BI dashboard built using Super Store Sales dataset.

## Features
- Sales Analysis
- Profit Analysis
- Orders Analysis
- Ship Days KPI
- Sales by Category
- Sales by Sub Category
- Sales by Region
- Sales by Segment
- Monthly Sales & Profit Trends
- Interactive Slicers

## Tools Used
- Power BI
- DAX
- Power Query

## Dataset
Super Store Sales Dataset
